# Cash Flow Tracker

A simple and functional **Salary & Expense Tracker** built using **HTML, CSS, and Vanilla JavaScript**.
This project demonstrates core concepts like DOM manipulation, event handling, and data persistence using localStorage.

## Features

* 💵 Set and display total salary
* ➕ Add expenses with name and amount
* 📋 Dynamic expense list (DOM manipulation)
* 🧮 Automatic calculation of:

  * Total Expenses
  * Remaining Balance
* ❌ Delete individual expenses
* 💾 Data persistence using localStorage
* 📊 Pie chart visualization using Chart.js
* 📱 Responsive design
